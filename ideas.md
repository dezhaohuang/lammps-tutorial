# LAMMPS 入门教学网站 — 设计方案头脑风暴

## 方案一：「科研笔记本」— Editorial / Academic Modernism

### 设计运动
学术出版物 × 现代编辑排版（Editorial Design），灵感来源于 Nature / Science 等顶级期刊的网页版面，以及 Notion 的简洁文档美学。

### 核心原则
1. **信息密度与呼吸感并存** — 大量教学内容需要清晰层次，但不能像 PDF 手册一样密不透风
2. **功能驱动装饰** — 每个视觉元素都服务于信息传达，不做纯装饰
3. **代码即主角** — 代码块是本站最核心的内容，必须在视觉上被尊重
4. **渐进式揭示** — 通过折叠、Tab、步骤卡片控制信息量

### 配色哲学
- 主色：深靛蓝 `#1e3a5f`（学术权威感）+ 科技青 `#0ea5e9`（活力、可信）
- 背景：暖白 `#fafaf8`（纸张质感，减少屏幕疲劳）
- 代码块：深灰底 `#1e293b` + 语法高亮
- 强调色：琥珀 `#f59e0b`（用于警告、重点标注）

### 布局范式
- 左侧固定目录导航（桌面端），移动端折叠为汉堡菜单
- 主内容区采用 **非对称双栏**：左侧文字说明，右侧代码/图示
- Hero 区使用全宽分子动力学可视化背景 + 毛玻璃文字层

### 签名元素
1. 「分子轨迹」装饰线 — 用 SVG 绘制的原子连接线作为分隔符
2. 「终端窗口」代码块 — 模拟真实终端的圆角窗口 + 标题栏 + 复制按钮
3. 步骤指示器 — 每个教程用数字圆圈 + 连接线标注进度

### 交互哲学
- 平滑滚动 + 目录高亮跟随
- 代码块 hover 显示复制按钮，点击后显示"已复制"反馈
- Tab 切换不同平台（Windows / macOS / Linux）
- 折叠面板用于 FAQ 和可选的高级内容

### 动画
- 页面滚动时内容块从下方淡入（stagger 效果）
- 代码块打字机效果（仅首次进入视口时）
- 目录导航的平滑高亮过渡
- 按钮 hover 的微妙上浮 + 阴影加深

### 字体系统
- 标题：Noto Serif SC（宋体衬线，学术感）
- 正文：Noto Sans SC（黑体无衬线，可读性）
- 代码：JetBrains Mono（等宽，专业感）

## 方案二：「实验室白板」— Swiss / Bauhaus Minimalism

### 设计运动
瑞士国际主义风格 × 包豪斯功能主义，灵感来源于 Stripe 文档和 Linear 的极简界面。

### 核心原则
1. **网格即秩序** — 严格的 8px 网格系统，所有元素对齐
2. **留白即奢侈** — 大量空白空间让内容呼吸
3. **色彩极度克制** — 几乎全灰度，仅用一个强调色
4. **排版即设计** — 通过字号、字重、间距创造层次，而非装饰

### 配色哲学
- 主色：纯黑 `#0f172a`（权威、专注）
- 强调色：电光蓝 `#3b82f6`（唯一的彩色，用于链接、按钮、进度）
- 背景：纯白 `#ffffff` + 极淡灰分区 `#f8fafc`
- 代码块：`#0f172a` 深色背景

### 布局范式
- 单栏为主，最大宽度 720px，居中排列
- 每个章节之间用大量留白分隔（80-120px）
- Hero 区极简：一行大标题 + 一行副标题 + 一个按钮

### 签名元素
1. 粗体数字标注 — 每个步骤用超大号数字（72px+）作为视觉锚点
2. 细线分隔符 — 1px 灰线作为章节分隔
3. 标签系统 — 小型圆角标签标注"Windows" "macOS" "Linux"

### 交互哲学
- 极简过渡，200ms ease-out
- 无多余动画，只有必要的状态反馈
- 键盘友好，Tab 可达所有交互元素
- 搜索功能突出，Cmd+K 快捷键

### 动画
- 几乎无动画，仅有：按钮 hover 背景色变化、代码复制反馈、页面切换淡入
- 哲学：动画是噪音，除非它传达信息

### 字体系统
- 标题：Inter（几何无衬线，瑞士风格代表）
- 正文：Inter
- 代码：Fira Code（连字等宽字体）

## 方案三：「粒子宇宙」— Sci-Fi / Computational Aesthetics

### 设计运动
科幻计算美学 × 数据可视化艺术，灵感来源于 Three.js 官网、CERN 的粒子物理可视化、以及赛博朋克 UI。

### 核心原则
1. **暗色即沉浸** — 深色背景让用户专注于内容，同时呼应"计算"氛围
2. **粒子即隐喻** — 分子动力学本身就是粒子运动，设计语言与学科内容统一
3. **渐变即能量** — 用渐变色表达粒子间的能量场和相互作用
4. **层叠即深度** — 多层半透明面板创造 Z 轴深度感

### 配色哲学
- 主背景：深空蓝 `#0a0e1a`（宇宙感）
- 主色：量子青 `#06b6d4` → 能量紫 `#8b5cf6` 渐变
- 卡片背景：`rgba(15, 23, 42, 0.8)` + backdrop-blur
- 代码块：更深的 `#050810` + 霓虹语法高亮
- 强调色：等离子橙 `#f97316`（用于警告、重要标注）

### 布局范式
- 全宽沉浸式布局，无明显边界
- Hero 区使用粒子动画背景（Canvas / CSS 动画）
- 内容区使用毛玻璃卡片浮在深色背景上
- 侧边导航使用半透明面板

### 签名元素
1. 粒子连接线背景 — 用 CSS 或 Canvas 绘制的动态粒子网络
2. 发光边框卡片 — 卡片边框带有微妙的渐变发光效果
3. 终端风格代码块 — 绿色/青色文字 + 闪烁光标 + 扫描线效果

### 交互哲学
- 鼠标移动触发粒子跟随效果
- 卡片 hover 时边框发光增强
- 滚动时背景粒子视差移动
- 代码块有"运行"按钮的视觉暗示

### 动画
- 背景粒子持续缓慢运动
- 内容块进入视口时从模糊到清晰的"聚焦"效果
- 按钮点击时的涟漪扩散效果
- 进度指示器的脉冲发光动画

### 字体系统
- 标题：Space Grotesk（几何感强，科幻气质）
- 正文：IBM Plex Sans（技术文档标准）
- 代码：IBM Plex Mono（与正文字体家族统一）

## 最终选择

**选择方案一：「科研笔记本」— Editorial / Academic Modernism**

理由：
1. 目标用户是研究生和科研人员，学术化的设计语言最契合受众心理预期
2. 非对称布局 + 左侧导航最适合长篇教程内容的阅读体验
3. 暖白背景 + 深色代码块的对比度最佳，长时间阅读不疲劳
4. 「终端窗口」风格的代码块既美观又实用，强化"可操作"的教学感
5. 方案三虽然视觉冲击力强，但暗色主题不适合长篇教学阅读；方案二过于克制，缺乏教学网站需要的引导感
